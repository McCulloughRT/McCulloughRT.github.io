GeoSpatial Analysis

Heatmapping development density with MapBox.js.

This project scrapes data about architecture projects in the City of Portland's development pipeline, converts them to GeoJSON, and plots them on a map with a heat overlay to show where patterns emerge. Projects can be filtered by a variety of factors.

Revit Warnings Project

Graphing the frequency of warnings in Autodesk Revit.

Understanding the magnitude of a problem is one of the first steps towards fixing it. Node.js is used server-side on AWS to parse warnings exports from Revit into a MongoDB database that track warning type and project phase. The site is served by express, and graphs are created with d3.js. There are currently over 20,000 warning entries from 5 different countries.

Intuitive Design Control

A project that was created so that we can design intuitively with our own hands, instead of trying to understand how to make the software acheive what we desire. Using a Leap Motion Controller, bone movements in the viewers hands are monitored and translated to camera movements.

Actions such as pinching and grabbing allow the designer to have real physical interaction with their designs. Additionally, an experimental WebVR module is included that will pipe graphics directly to an Oculus Rift device.

Algorithmic Design 1

Parametric design with Dynamo.

Visual programming is used to create a fabric like sculpture. Control modules are created to allow flexing of armature along the fabric, control falloff rates from flexed positions, and introduce random organic noise.

Algorithmic Design 2

Parametric design with Dynamo.

Visual programming is used to interpret two-dimensional abstract paintings into multi-million point three-dimensional models. All parts of the model are brought automatically into Autodesk Revit with position and construction data for simple fabrication.
